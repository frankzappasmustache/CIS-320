Tags used to create this page

  • Nesting:
    This tag is an example of nesting. I have placed an h5 header inside of a list item inside of an unordered list.
  • Citation Markup:
    This is a citation. It is used to markup the title of a cited work.
  • Abbreviation Tag:
    abbr is a tag used to abbreviate something and show the fully expanded word upon mouseover
  • Quotes:
    A quote tag is used to distinguish a word or sentence from the rest of a paragraph in html by using quotation marks.
  • Links:
    The link tag "a" is used to link to a source outside of a document or at a different anchored place in the document. This short paragraph is a link to the w3 schools page on html links.

Update with unordered list of attributes used in implementing images lab along with their roles

  • src:
    The src (source) attribute is used to specify the URL of an external resource.
  • alt:
    The alt (alternative) attribute gives the user alternative information for an image in the case that they cannot see the image.
  • style:
    The style attribute is used to define style for a specific html element, and it overrides any global styles that are set by the <style> tag or any css files.

All three of these images are from another server, and to display them I simply included the image's URL in the src attribute. If I were to use an image hosted on the same server but in a different folder, I would simply include the full path to the file (e.g. /path/to/image.png).
If someone using a screenreader were to view this page, they would hear the text included in the alt attribute for each image, which tells them the comic author and comic name. It is important that alt text is included for people who may be blind or not able to read the page for other reasons. The alt text provides a description of the image tha can be read to them for context.
